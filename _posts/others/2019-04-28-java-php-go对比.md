## go介绍
###go特点

- go可以不加分号
- 没有继承没有多态
- 没有异常处理
- 没有public private 等属性，首字母大写公有 ，小写私有
- 导出包 变量 没有使用会编译报错

### go优点（开发快，运行快，部署快）
优点
- 编译速度快
- 开发效率高接近php，效运行率高接近c语言
- 学习成本底
- 组合思想，无侵入式编程，天生的并发编程支持，支持所有编程的范式比如面向对象，过程，面向接口
- 部署方便 二进制 copy方式
- 高度支持并发
- 稳定性强
- 标准库强大 应用层 网络层 系统层支持很好
- 稳定性好

### 特点总结
- 静态语言；天生并发；内置GC（垃圾回收）；语法简单；开发快；运行快；部署快
